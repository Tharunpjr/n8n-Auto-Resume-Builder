# AI Resume Automation Workflow (n8n)

This n8n workflow automates the process of:
- Scraping LinkedIn job postings (AI/ML roles)
- Matching suitability using Google Gemini AI
- Generating an enhanced resume in Google Docs
- Sending the result via Gmail and Telegram

**Tech Stack:** n8n, Google Docs API, Gmail API, Telegram Bot, LangChain, Google Gemini

To use:
1. Import `ai resume.json` into n8n.
2. Connect your Google and Telegram credentials.
3. Run the workflow to auto-generate tailored resumes daily.
